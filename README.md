***手搓一条类龙虾***（完整版请查看pdf）

最近养龙虾大火，突发奇想，龙虾到底是什么，有没有可能手搓一条。跟LLM沟通了一下想法，发现完全可行的，龙虾到底是什么呢？

一、首先是一个智能体agent，不单是一个问答机器人，而是一个会做事的机器人。怎么实现这个功能？就是通过本地的功能模块封装（skills），这些封装都是简单的专项能力，并且不是独立完成，而是驱动或者调用LLM来完成大部分工作。也就是一套协同LLM工作的机制，当然Agent需要实现长记忆，避免上言不接下语和幻觉。

二、Gateway (网关)​：为了能跟聊天机器人交付，要按照飞书（以飞书为例）等机器人实现必要的互动，包括从飞书app发出机器人操作要求，并处理返回回答。

三、路由：为了节约token，根据问题难易程度，调用成本不一样的LLM。避免小问题问贵模型。

经过LLM分析和真人消化，发现难度也不大，有Deepseek没问题。
